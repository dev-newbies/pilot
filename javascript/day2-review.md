# 값(데이터)의 종류 - 데이터 타입
- 뭔가 많았던 것 같지만 줄여서 살펴봅시다.
- 기본(원시, Primitive) 타입 - 참과 거짓, 정수, 소수점, 문자열
- 기본 타입들을 조합해서 데이터의 덩어리인 객체(Object)
    - 객체로 표현해 보기: 객체로 표현하고자 하는 것이 어떤 속성을 가지고 있는지 하나씩 표현해 봅시다.
        - 형광등
        - TV
        - 책
        - 자동차
        - 생각나는 것 하나
        - 같이 해 봅시다.
            - 컴퓨터 화면과 마우스의 위치
            - 도서관
    - 객체로 표현한 것의 동작이 무엇이 있을 지 생각해보기
